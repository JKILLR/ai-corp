# Internal Communications Skill

## Overview

This skill provides guidance for writing effective internal communications within AI Corp.

## Communication Types

### Status Reports
- Lead with key outcomes and blockers
- Use bullet points for clarity
- Include metrics where applicable
- Keep it concise (aim for 1-2 paragraphs max)

### Announcements
- Clear subject line
- Context: why this matters
- What's changing
- Action items (if any)
- Timeline

### Documentation
- Start with a summary
- Use headers for navigation
- Include examples
- Keep it up to date

## Tone Guidelines

- Professional but approachable
- Clear and direct
- Action-oriented
- Avoid jargon unless necessary

## Templates

Use the templates in the `templates/` directory for common communication types.
